TVDex Discord App - Privacy Policy

The following document explains what data is collected by TVDex.

Glossary

  • “Application” or “Discord application” refers to the Discord app user, its associated owners and the servers it has.
  • “Data” refers to data stored by an individual instance, including personal data.
  • “App” or “instance” refers to the copy of the code running the Discord application, with its own data.
  • “Collectible” refers to any object that is meant to be collected, such as drivers or other future means like virtual currency.
  • “Application owners” refers to the owners of the Discord application, meaning the one with any form of access to the application's authentication method and data belonging to the instance.
  • “TVDex moderation team” or “moderation team” refers to the owners of the Discord application and the users they may chose to be part of the moderation team. You may find a list of these users on the Discord server.

What data is collected

TVDex collects the following data from Discord:

  • User IDs, this is used to identify you inside our database.
  • Server (or guild) IDs, this is used to store settings necessary to the App's operation.
  • Channel IDs, this is used to know in which channel should the collectibles spawn.

In addition, the following data proper to TVDex is created and used:

  • The list of collectibles owned by a user of the service.
  • A history of trades done on collectibles, including the users that once owned the said collectible but do not anymore.

How the data is stored

All data is stored on a PostgreSQL server, running on a Virtual Private Machine hosted by Oracle Corportation.

Interaction between the App and the database server is exclusively local using Tortoise ORM.

Access to the data

The only persons allowed to access the data are the application owners.

Any application owner may interact with the App and the administrator interface. Only laggron42 has direct access on the host machine.

The TVDex moderation teams do not have access to the data.

The data may never be made available to the public, and its access must be secured accordingly.

Your rights

You may request a copy or the deletion of your personal data held by the application by contacting the application owners in the Discord server.
